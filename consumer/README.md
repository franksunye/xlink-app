# 修链房屋维修 - 消费者端

修链房屋维修是一款基于 **uni-app** 开发的房屋维保服务平台小程序，旨在为用户提供一站式的家庭维修、保养及翻新服务。本项目为消费者端，支持微信小程序、支付宝小程序等多个平台。

## 🚀 项目概览

- **项目名称**：修链房屋维修
- **技术栈**：Vue 2.x, uni-app, uni-ui
- **主要目标**：实现便捷的报修流程、透明的价格体系以及全方位的维保服务管理。

## ✨ 主要功能

- **🏠 首页展示**：精选服务推荐、热销商品展示及活动预热。
- **🔍 服务分类**：清晰的服务目录，涵盖各类房屋维修、清洁及安装服务。
- **⚡ 快速下单**：支持一键报修，通过定位及详情描述快速匹配师傅。
- **👥 专属管家**：提供管家式服务，查看管家信息及进度跟踪。
- **📜 订单管理**：全流程订单状态追踪（派单、施工、验收、结算）。
- **💰 收益与奖励**：分享提成、我的奖励、提现明细等营销激励功能。
- **📑 合同与质保**：电子合同在线签署、在线验收单及电子质保单查看。
- **🎫 优惠券中心**：卡券中心及使用记录，提供多样的促销体验。
- **🧮 报价计算器**：透明的价格预估工具。

## 📂 项目结构

```text
├── common/             # 公共 JS 工具类（高德地图 SDK、公共逻辑等）
├── components/         # 自定义组件
├── custom-tab-bar/     # 微信小程序自定义 TabBar 实现
├── customize-tab-bar/  # 支付宝小程序自定义 TabBar 实现
├── pages/              # 主包页面
│   ├── index/          # 首页
│   ├── classification/ # 分类页
│   ├── declaration/    # 快速报修/下单
│   ├── message/        # 客服消息
│   └── account/        # 个人中心
├── subpackages/        # 分包页面（订单、商品详情、管家、奖励等）
├── static/             # 静态资源（图片、图标）
├── scripts/            # 项目构建或辅助脚本
├── uni_modules/        # uni-ui 插件及第三方模块
├── wxcomponents/       # 微信原生组件（用于原生插件集成）
├── App.vue             # 应用配置
├── API.md              # 接口文档说明
├── manifest.json       # 项目配置（AppID、权限等）
└── pages.json          # 页面路由及窗口样式配置
```

## 🛠 开发与运行

1. 下载并安装 [HBuilderX](https://www.dcloud.io/hbuilderx.html)。
2. 将本项目拖入 HBuilderX 中。
3. 点击上方菜单栏：`运行` -> `运行到小程序模拟器` -> `微信开发者工具` (确保已安装微信开发者工具并开启服务端口)。
