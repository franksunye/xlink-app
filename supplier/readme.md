**1. 项目简介**  
  1. 本项目为FSGO前端，框架使用了uni-app（基于Vue.js开发，编写一套代码，可发布到iOS、Android、H5、以及各种小程序、快应用等多个平台)、组件库使用了vant-weap、封装的公共方法在"/resources/js/common.js"。

**2. 安装**
  1. [下载HBuilderX开发工具](https://www.dcloud.io/hbuilderx.html)（下载App开发版，可开箱即用；如下载标准版，在运行或发行uni-app时，会提示安装uni-app插件，插件下载完成后方可使用。）
  2. 在HBuilderX菜单栏点击 "文件 -> 导入项目"（本地或svn://www.svn999.com/lidawei.fsgo_dev/code/app/uniapp）
  3. 在HBuilderX菜单栏点击 "工具 -> 插件安装"，选择安装“scss/sass编译”

**3. 运行**（[详见uniapp官方文档](https://uniapp.dcloud.io/)）
  1. 菜单栏点击“运行”，选择要运行的端

**4. 发行**（[详见uniapp官方文档](https://uniapp.dcloud.io/)）
  1. 菜单栏点击“发行”，选择要发布的端
